# Miku

🚀 A lightweight React-like framework built from scratch with fiber-based reconciliation and concurrent rendering.

## What is Miku?

An educational implementation of modern frontend framework concepts including virtual DOM reconciliation, fiber-based rendering, and concurrent rendering with time slicing. Built entirely in TypeScript with JSX support.

**Status:** Work in progress - architecture and features are actively being developed and improved.

## Quick Start

```bash
npm install
npx tsc
# Open index.html in browser
```
